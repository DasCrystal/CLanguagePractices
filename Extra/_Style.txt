迴圈

    以x為小括弧內的變數。

    for迴圈
	小括弧內
            與遍歷(包含陣列)有關的，從0開始算，以 「< 元素個數」為結尾。
            與計數有關的，從1開始算，以「<= 次數」為結尾。

    雙層迴圈
        外層用y，內層用x。(與座標有關)

    結尾表達式
        用 var += 1 表示，而不是var++。

    for (int y = 1; y <= ArrayLen; y += 1) {

        for (int x = 1; x <= y; x += 1) {

            /* 內容 */

        }

    }

空行

    程式區塊內的第一行與最後一行通常應該保持空白，
    但 return & break & continue 應該寫在區塊內的最後一行。

    int main() {

        /* 內容 */

        return 0;
    }

符號
    
    左大括弧應該放在描述程式區塊屬性的語句(像聲明或表達式)後面，
    而不是換新行後才放左大括弧。

    struct EFI_SYSTEM_TABLE { // 註解

        /* 內容 */

    }

陣列

    數列的第零項元素必須儲存陣列本身的長度或其他有關於陣列本身的資訊(類型ID之類的)
